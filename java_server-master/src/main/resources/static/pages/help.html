Help

  • In the add operation, if you need to manually enter the id, please try to use the id that has never been used, do not use the id of the deleted record, because in order to maintain data integrity, some records in the past were not really deleted, just set you are invisible. So when it is prompted that the id already exists, don't be troubled by the fact that it is not in the table.
  • Only orders in the past two weeks are displayed for pending orders and orders in delivery, and only orders within two months are displayed for completed orders.
